# Changelog

All notable changes to `aware-api` are documented here. Dates use UTC.

## [0.2.0] - 2026-05-10
- Removed the legacy high-level Node/Environment client, context DTOs, function-call models, and network-operation builders.
- Kept the package focused on generated-client invocation manifests, interface specs, and transport-neutral endpoint invocation.

## [0.1.0] - 2025-10-13
- Initialized project structure (pyproject, README, tests scaffold).
- Added the first experimental high-level API client implementation.
